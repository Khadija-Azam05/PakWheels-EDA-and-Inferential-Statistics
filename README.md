# Inferential-Statistics-and-EDA

This project focuses on **Exploratory Data Analysis (EDA)** and **Inferential Statistics** using Python.  
The assignment includes statistical analysis, hypothesis testing, correlation analysis, bootstrapping, and visualization using real-world datasets such as the PakWheels automobile dataset and spurious correlation datasets.

# 📌 Project Overview

The objective of this project is to demonstrate practical understanding of:
- Descriptive statistics
- Inferential statistics
- Hypothesis testing
- Correlation analysis
- Bootstrapping
- Data visualization
- Statistical interpretation

The project emphasizes both technical implementation and interpretation of statistical findings.
